\doxysubsection{Detailed Description}
Abstract base class for setting data sources. 

\doxylink{class_f_gameplay_setting_data_source}{FGameplay\+Setting\+Data\+Source} provides an abstraction layer for getting and setting values from various sources. This allows settings to read/\+write data from:\+
\begin{DoxyItemize}
\item Configuration files (ini files)
\item User settings objects
\item Console variables
\item Custom data sources
\end{DoxyItemize}

Data sources use a string-\/based interface for maximum flexibility, allowing derived classes to convert between string representations and their native types.

\begin{DoxyNote}{Note}
This is an abstract class. Use \doxylink{class_f_gameplay_setting_data_source_dynamic}{FGameplay\+Setting\+Data\+Source\+Dynamic} or implement your own. 
\end{DoxyNote}
