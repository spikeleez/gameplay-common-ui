\doxysubsection{Detailed Description}
Root widget for managing the primary gameplay UI layout. 

This widget is responsible for coordinating the standard HUD, handling the "{}\+Escape"{} action to open menus, and managing the "{}\+Controller Disconnected"{} logic for the local player.
