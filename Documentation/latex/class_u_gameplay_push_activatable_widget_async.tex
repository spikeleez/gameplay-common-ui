\doxysubsection{Detailed Description}
Delegate for when a widget is about to be pushed or has been pushed. 

Async action to load and push a widget to a specific UI layer

This action handles the asynchronous loading of a widget (either by class or by tag) and then pushes it onto a designated layer within the \doxylink{class_u_gameplay_primary_layout}{UGameplay\+Primary\+Layout}.
