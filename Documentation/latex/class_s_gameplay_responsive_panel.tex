\doxysubsection{Detailed Description}
Slate implementation of a panel that can wrap its content based on available space. 

\doxylink{class_s_gameplay_responsive_panel}{SGameplay\+Responsive\+Panel} acts as a wrapper around an SGrid\+Panel, dynamically shifting child widgets between rows and columns depending on the screen size and configuration.
