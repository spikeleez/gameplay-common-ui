\doxysubsection{Detailed Description}
Base class for widgets that can be activated and deactivated within the UI stack. 

Provides enhanced support for input configuration (Menu vs Game vs Both) and automated binding/\+unbinding of UI actions based on the widget\textquotesingle{}s lifecycle.
