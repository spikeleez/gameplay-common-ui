\doxysubsection{Detailed Description}
Represents settings that are shared across different platforms or machines for a single user. 

Unlike \doxylink{class_u_gameplay_settings_local}{UGameplay\+Settings\+Local} which stores machine-\/specific performance settings, \doxylink{class_u_gameplay_settings_shared}{UGameplay\+Settings\+Shared} is intended for gameplay-\/affecting preferences that would ideally travel with the user\textquotesingle{}s account (via Save\+Game).

Examples:\+
\begin{DoxyItemize}
\item Subtitle preferences (Enabled, Text Size, Color, Opacity)
\item Language/\+\+Culture preferences
\item Accessibility settings 
\end{DoxyItemize}
