\doxysubsection{Detailed Description}
Standard UI widget for modal confirmation and error dialogs. 

Works in tandem with \doxylink{class_u_gameplay_confirmation_descriptor}{UGameplay\+Confirmation\+Descriptor} to dynamically build buttons and display messages. Broadcasts the user\textquotesingle{}s choice via a callback when a button is clicked or the dialog is dismissed.
