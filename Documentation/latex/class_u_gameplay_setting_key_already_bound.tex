\doxysubsection{Detailed Description}
Feedback widget shown when a user tries to bind a key that\textquotesingle{}s already in use. 

This widget extends the "{}\+Press Any Key"{} flow to include specific feedback about binding conflicts, allowing the user to see what they are about to overwrite.
