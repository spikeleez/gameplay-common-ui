\doxysubsection{Detailed Description}
Data source that uses reflection (Property Paths) to access data. 

\doxylink{class_f_gameplay_setting_data_source_dynamic}{FGameplay\+Setting\+Data\+Source\+Dynamic} allows settings to dynamically read and write to any UObject property that can be reached via a property path from the Local\+Player or related objects.
