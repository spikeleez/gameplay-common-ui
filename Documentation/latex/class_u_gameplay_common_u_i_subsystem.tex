\doxysubsection{Detailed Description}
Core Game\+Instance subsystem for managing the global UI lifecycle. 

This subsystem acts as the central coordinator for the UI system. It initializes the configured UI policy, manages player connection/\+disconnection events, and provides global accessibility to the active UI policy.
