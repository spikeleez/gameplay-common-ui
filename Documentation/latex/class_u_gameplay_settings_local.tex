\doxysubsection{Detailed Description}
Manages local, non-\/replicated game settings for the user. 

This class extends UGame\+User\+Settings to add project-\/specific settings like:\+
\begin{DoxyItemize}
\item Complex frame rate limiting (Menu, Background, Battery)
\item Display gamma
\item Performance stat HUD visibility
\item Latency tracking (Reflex)
\item Input controller branding 
\end{DoxyItemize}
