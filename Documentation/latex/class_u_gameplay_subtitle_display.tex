\doxysubsection{Detailed Description}
UWidget wrapper for displaying subtitles in UMG. 

Synchronizes with the Gameplay\+Subtitles\+Subsystem to automatically display subtitle text. Supports dynamic styling via \doxylink{struct_f_gameplay_subtitle_format}{FGameplay\+Subtitle\+Format} and \doxylink{class_u_gameplay_subtitle_display_options}{UGameplay\+Subtitle\+Display\+Options}.
