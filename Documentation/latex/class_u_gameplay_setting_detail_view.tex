\doxysubsection{Detailed Description}
Widget that displays detailed information and extended controls for a setting. 

The detail view typically appears when a setting is hovered or selected. It shows:\+
\begin{DoxyItemize}
\item Setting Name
\item Description (Rich Text)
\item Dynamic details (e.\+g. current value vs base value)
\item Warnings or error messages
\item Disabled reasons
\item Any registered Detail Extensions (specialized sub-\/widgets)
\end{DoxyItemize}

It uses a widget pool to efficiently manage extension widgets.
