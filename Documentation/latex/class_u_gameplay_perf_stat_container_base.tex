\doxysubsection{Detailed Description}
Widget container that manages multiple performance statistic widgets. 

This panel is responsible for aggregating various performance stat widgets (like FPS, Ping, etc.) and controlling their visibility globally based on the user\textquotesingle{}s performance settings.
