\doxysubsection{Detailed Description}
A container widget that allows child widgets to flow horizontally and wrap vertically. 

\doxylink{class_u_gameplay_responsive_panel}{UGameplay\+Responsive\+Panel} is ideal for settings screens where some categories might need to stack on smaller screens or mobile devices while staying horizontal on desktop/\+widescreen displays.
