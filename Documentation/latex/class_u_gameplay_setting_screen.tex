\doxysubsection{Detailed Description}
Top-\/level activatable widget for a full settings menu screen. 

\doxylink{class_u_gameplay_setting_screen}{UGameplay\+Setting\+Screen} provides the high-\/level orchestration for a settings menu, including:\+
\begin{DoxyItemize}
\item Creating and managing the \doxylink{class_u_gameplay_setting_registry}{UGameplay\+Setting\+Registry}
\item Tracking change state (dirty/\+applied) via \doxylink{class_f_gameplay_setting_registry_change_tracker}{FGameplay\+Setting\+Registry\+Change\+Tracker}
\item Handling global actions like Apply, Cancel, and Reset
\item Input action mapping for standard settings operations
\item Managing the underlying UGameplay\+Setting\+Panel 
\end{DoxyItemize}
