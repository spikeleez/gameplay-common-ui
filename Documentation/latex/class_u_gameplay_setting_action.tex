\doxysubsection{Detailed Description}
Represents a setting that performs an action when activated. 

\doxylink{class_u_gameplay_setting_action}{UGameplay\+Setting\+Action} is a special type of setting that doesn\textquotesingle{}t store a value but instead triggers an action when the user interacts with it. Actions can be:\+


\begin{DoxyItemize}
\item {\bfseries{Named Actions}}:\+ Using Gameplay\+Tags to identify specific actions (e.\+g., "{}\+Settings.\+Show\+Credits"{})
\item {\bfseries{Custom Actions}}:\+ Binding custom functions for specialized behavior
\end{DoxyItemize}

Common uses include:\+
\begin{DoxyItemize}
\item Opening supplementary screens (Credits, EULA, Privacy Policy)
\item Triggering game-\/specific commands (Screenshot, Reset Controls)
\item Executing confirmation dialogs
\end{DoxyItemize}

By default, actions don\textquotesingle{}t mark settings as changed (dirty), but this can be configured for actions that modify persistent state.
