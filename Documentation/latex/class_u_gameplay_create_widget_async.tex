\doxysubsection{Detailed Description}
Delegate for when the async widget creation is complete. 

Async action to load and create a widget

This action handles the asynchronous loading of a widget class and its subsequent instantiation. It can optionally suspend player input while the loading is in progress to prevent interaction during transitions.
