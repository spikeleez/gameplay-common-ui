\doxysubsection{Detailed Description}
Base class for widgets displaying a single performance statistic. 

\doxylink{class_u_gameplay_perf_stat_widget_base}{UGameplay\+Perf\+Stat\+Widget\+Base} handles common functionality for displaying performance metrics like FPS, Ping, or Packet Loss. It supports showing a live graph if a \doxylink{class_u_gameplay_perf_stat_graph}{UGameplay\+Perf\+Stat\+Graph} is provided in the widget hierarchy.
