\doxysubsection{Detailed Description}
Button widget that automatically displays the icon/\+text for a bound UI action. 

This button is typically used in footers or legends to show the player which key or button is currently bound to a specific menu action (e.\+g., "{}\+Back"{}, "{}\+Accept"{}).
