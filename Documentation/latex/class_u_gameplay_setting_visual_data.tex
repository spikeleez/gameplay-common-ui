\doxysubsection{Detailed Description}
Data asset that maps settings to their visual representations in the UI. 

\doxylink{class_u_gameplay_setting_visual_data}{UGameplay\+Setting\+Visual\+Data} is used to decide:\+
\begin{DoxyEnumerate}
\item Which UUser\+Widget (List\+Entry) to use for a particular setting in a list.
\item Which additional widgets (Detail\+Extensions) to show for a setting in the detail pane.
\end{DoxyEnumerate}

Mapping can be done by either the setting\textquotesingle{}s C++ class or its unique FName identifier.
