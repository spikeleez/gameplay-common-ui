\doxysubsection{Detailed Description}
Tracks changes made to settings within a registry. 

\doxylink{class_f_gameplay_setting_registry_change_tracker}{FGameplay\+Setting\+Registry\+Change\+Tracker} listens for change events from a registry and keeps track of which settings have been modified. It provides functionality to apply all pending changes, restore modified settings to their initial values, and check if any settings are currently "{}dirty"{} (changed but not applied).
