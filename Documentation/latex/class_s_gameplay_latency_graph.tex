\doxysubsection{Detailed Description}
Slate widget for rendering a latency/\+performance graph. 

\doxylink{class_s_gameplay_latency_graph}{SGameplay\+Latency\+Graph} is a low-\/level Slate widget designed to efficiently render a line graph of performance data (like frame times) using the engine\textquotesingle{}s rendering context. It supports background color and line color customization.
