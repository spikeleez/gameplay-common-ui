\doxysubsection{Detailed Description}
Represents the current filter configuration for a settings screen. 

\doxylink{struct_f_gameplay_setting_filter_state}{FGameplay\+Setting\+Filter\+State} is used to determine which settings should be visible and available in the UI. It supports filtering by:\+
\begin{DoxyItemize}
\item Search text (using FText\+Filter\+Expression\+Evaluator)
\item Explicit allow lists
\item Visibility/\+\+Enabled state
\item Resettable state
\item Inclusion of nested pages 
\end{DoxyItemize}
