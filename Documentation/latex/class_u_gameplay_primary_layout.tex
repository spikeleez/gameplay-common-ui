\doxysubsection{Detailed Description}
The root UI layout for a single player. 

This widget class acts as the container for all UI layers (HUD, Menus, Modals) for a specific player. It manages the registration of these layers and provides methods to push/\+pop activatable widgets onto specific stacks.
