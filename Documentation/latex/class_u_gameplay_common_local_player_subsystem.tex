\doxysubsection{Detailed Description}
Subsystem managing UI-\/related state for a specific local player. 

This subsystem handles player-\/specific UI logic, such as showing confirmation or error dialogs, and tracking changes to the player\textquotesingle{}s pawn and player state for UI updates.
