\doxysubsection{Detailed Description}
Stores a buffer of performance stat samples and provides statistical analysis. 

\doxylink{class_f_gameplay_sampled_stat_cache}{FGameplay\+Sampled\+Stat\+Cache} maintains a ring buffer of statistical samples and provides utilities to calculate minimum, maximum, and average values over the sample period.

This class is useful for tracking performance metrics over time and generating charts or visualizations of performance data.

\begin{DoxyNote}{Note}
Uses a ring buffer for efficient memory usage 
\end{DoxyNote}
