\doxysubsection{Detailed Description}
Base class for widgets that extend the functionality of a setting\textquotesingle{}s detail view. 

Extensions are auxiliary widgets that provide additional UI elements or information for a specific setting (e.\+g., a "{}\+Restore Default"{} button, a descriptive image, or a specialized control).
