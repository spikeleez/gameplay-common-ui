\doxysubsection{Detailed Description}
Subsystem for managing global subtitle display settings. 

This subsystem maintains the current formatting and display preferences for subtitles (size, color, background, etc.) across the game instance. It broadcasts events when these settings change so that subtitle widgets can update accordingly.
