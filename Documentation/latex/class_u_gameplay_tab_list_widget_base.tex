\doxysubsection{Detailed Description}
Extension of the common tab list with support for dynamic registration and visibility. 

This widget manages a list of tabs, usually corresponding to pages in a switcher. It supports pre-\/registered tabs defined in the editor and dynamic tabs added at runtime.
