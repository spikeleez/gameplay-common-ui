Detailed Description

Observer that caches performance stats for each frame.

FGameplayPerformanceStatCache implements IPerformanceDataConsumer to receive and cache performance data from the engine's charting system. It maintains a cache of sampled stat data for various performance metrics.

See also
IPerformanceDataConsumer, FGameplaySampledStatCache
