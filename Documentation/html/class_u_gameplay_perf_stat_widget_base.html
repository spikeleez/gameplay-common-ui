Detailed Description

Base class for widgets displaying a single performance statistic.

UGameplayPerfStatWidgetBase handles common functionality for displaying performance metrics like FPS, Ping, or Packet Loss. It supports showing a live graph if a UGameplayPerfStatGraph is provided in the widget hierarchy.
