Detailed Description

Represents the current filter configuration for a settings screen.

FGameplaySettingFilterState is used to determine which settings should be visible and available in the UI. It supports filtering by:

  • Search text (using FTextFilterExpressionEvaluator)
  • Explicit allow lists
  • Visibility/Enabled state
  • Resettable state
  • Inclusion of nested pages
