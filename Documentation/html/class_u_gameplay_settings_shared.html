Detailed Description

Represents settings that are shared across different platforms or machines for a single user.

Unlike UGameplaySettingsLocal which stores machine-specific performance settings, UGameplaySettingsShared is intended for gameplay-affecting preferences that would ideally travel with the user's account (via SaveGame).

Examples:

  • Subtitle preferences (Enabled, Text Size, Color, Opacity)
  • Language/Culture preferences
  • Accessibility settings
