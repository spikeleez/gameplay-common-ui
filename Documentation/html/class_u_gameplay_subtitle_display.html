Detailed Description

UWidget wrapper for displaying subtitles in UMG.

Synchronizes with the GameplaySubtitlesSubsystem to automatically display subtitle text. Supports dynamic styling via FGameplaySubtitleFormat and UGameplaySubtitleDisplayOptions.
