Detailed Description

Stores a buffer of performance stat samples and provides statistical analysis.

FGameplaySampledStatCache maintains a ring buffer of statistical samples and provides utilities to calculate minimum, maximum, and average values over the sample period.

This class is useful for tracking performance metrics over time and generating charts or visualizations of performance data.

Note
Uses a ring buffer for efficient memory usage
