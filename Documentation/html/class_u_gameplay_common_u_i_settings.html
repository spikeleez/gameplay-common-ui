Detailed Description

Global developer settings for the Gameplay Common UI system.

This class stores project-wide configurations that define which UI policy class to use, mappings for tags to activatable widgets, and default classes for confirmation dialogs. These settings are accessible via the Project Settings menu in the Unreal Editor.
