Detailed Description

Policy class that manages the UI lifecycle and primary layout.

The UI Policy is responsible for determining which primary layout class to use, creating it for local players, and managing its addition/removal from the viewport. It acts as the bridge between the GameplayCommonUISubsystem and the actual widget instances.
