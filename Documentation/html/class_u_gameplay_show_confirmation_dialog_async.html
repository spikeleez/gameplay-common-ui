Detailed Description

Delegate fired when the user selects an option in the confirmation dialog.

Async action to display a confirmation dialog

This class provides several static methods to quickly trigger different standard confirmation dialogs (Yes/No, Ok/Cancel, etc.) from Blueprints and wait for the user's choice.
