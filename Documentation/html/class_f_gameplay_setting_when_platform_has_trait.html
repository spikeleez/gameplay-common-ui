Detailed Description

Edit condition that evaluates based on platform-specific gameplay tags (traits).

This class checks if the current platform has or lacks a specific gameplay tag. Based on this check, it can either "Kill" (hide/immutable/no-analytics) or "Disable" the setting.

Commonly used for feature toggles that depend on platform capabilities.
