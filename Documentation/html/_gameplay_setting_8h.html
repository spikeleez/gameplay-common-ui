enum class EGameplaySettingChangedReason : uint8
strong

Specifies why a gameplay setting value changed.

This enum is used to track the reason for setting changes, which can be useful for analytics, UI updates, and understanding user behavior.

Enumerator
Change 

The setting was changed by the user or programmatically.

DependencyChanged 

The setting changed because a dependency changed.

ResetToDefault 

The setting was reset to its default value.

RestoreToInitial 

The setting was restored to its initial value.
