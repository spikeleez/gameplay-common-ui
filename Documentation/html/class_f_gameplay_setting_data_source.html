Detailed Description

Abstract base class for setting data sources.

FGameplaySettingDataSource provides an abstraction layer for getting and setting values from various sources. This allows settings to read/write data from:

  • Configuration files (ini files)
  • User settings objects
  • Console variables
  • Custom data sources

Data sources use a string-based interface for maximum flexibility, allowing derived classes to convert between string representations and their native types.

Note
This is an abstract class. Use FGameplaySettingDataSourceDynamic or implement your own.
